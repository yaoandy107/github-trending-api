# GitHub Trending API
An API that crawl GitHub trending page.

## Setup
```sh
git clone git@github.com:yaoandy107/github-trending-api.git
cd github-trending-api
npm i
```

After setup, you can start API server:
```sh
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
